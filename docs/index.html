Welcome

Oh-My-PickleDB is a lightweight and simple key-value store. It is built upon Python's json module and was inspired by redis. It is licensed with the BSD three-clause license.

Oh-My-PickleDB is fun

>>> from my_pickledb import PickleDB

>>> db = PickleDB("test.db", load=True, auto_dump=True)

>>> db.set('key', 'value')
>>> True

>>> db.get('key')
>>> value

>>> db.dump()
>>> True

Easy installation

$ pip install oh-my-pickledb
$ git clone https://github.com/tory1103/oh-my-pickledb.git
